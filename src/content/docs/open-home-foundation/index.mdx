## Assets

The logo is available in two approved lockups: the main version and a stacked version. These variations are designed to accommodate different applications and use cases.

### Main Lockup

When in doubt, use this, as it is the default and official version, certified and with the trademark registration. Prioritize this version in all brand collateral.
