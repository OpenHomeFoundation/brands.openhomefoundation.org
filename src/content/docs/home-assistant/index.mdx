## Assets

The logo is available in two approved lockups: a main version and a stacked version. These variations are designed to accommodate different applications and use cases.

When in doubt, refer to the [logo](/home-assistant/logo/) page for guidance on correct usage.

### Main Lockup

When in doubt, use this, as it is the default and official version, certified and with the trademark registration. Prioritize this version in all brand collateral.

### Logomark

In most instances, the logomark by itself is used as a social media icon, or for sharing with third parties when they want to publish with the logomark only – however,  this must only be done when the context of the logo is clear, for example it is accompanied with the name.
